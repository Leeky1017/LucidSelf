## 协作指南

### 精校新条目

1. 确定要精校的卷次，打开对应文件
2. 在该卷文件末尾追加新条目的L1+L2内容
3. 同步在 `data/semantics/jzyh_extracted_batch1.jsonl` 追加L2语义记录

### 修正已有条目

1. 直接在对应卷文件中定位并修改
2. 如涉及L2语义，同步更新JSONL文件

### 注意事项

- **不跨卷操作**：每次只编辑一个卷文件
- **L2数据分离**：Markdown文件只含L1+L2文本，结构化L2数据存于JSONL
- **保持一致性**：条目编号、semantic_id命名遵循既有规范
