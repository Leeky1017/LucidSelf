### 4.1 芝草

#### 原文（断句）

- 【史】汉武帝元封二年，诏甘泉宫中产芝九茎连叶，作芝房之歌。（《前汉书》）芝房歌云：「斋房产草，九茎连叶，宫童效异，披图按牒。玄气之精，回复此都，蔓蔓日茂，芝成灵华。」（《礼乐志》）
- 宣帝神爵元年，诏金芝九茎产于函德殿铜池中。（《本纪》）
- 明帝永平十七年，芝草生殿前。（《后汉书》）
- 章帝建初三年，零陵献芝草；五年，同。（《本纪》）章和元年，诏芝草之类岁月不绝。（《同上》）
- 顺帝时，临城县生紫芝。（《宋书》）
- 武帝时，襄阳获紫芝。（《齐书》）
- 隋文帝开皇十九年，朱雀等十六门生芝草。（《隋书》）
- 唐太宗贞观十七年，安礼门御榻产灵芝五茎，又太子寝室中产紫草十四茎，并为龙兴凤翥之形，又滁州言所部川源遍生芝草。（《旧唐书》）
- 玄宗天宝中，有玉芝产于大同殿之柱础，一本两茎，神光照于殿。（《本纪》）
- 肃宗上元中，延英殿御座生玉芝一茎三花，御制玉灵芝诗。（《本纪》）
- 张九龄母丧，有紫芝产座侧。（《本传》）

- 【传记】王者慈仁，则紫芝生，食之令人延年。（《孙氏瑞应图》）
- 王者宠近耆老养有道，则芝荚生。（《同上》）
- 摇光则陵山玄芝生。（《春秋运斗枢》）
- 王者德至草木，则芝草生。（《孝经援神契》）
- 汉桓帝时，芝草生中黄藏府。（《东观汉记》）
- 唐贞观十九年，李树生芝英，赤盖紫光，色鲜丽。（《唐春秋》）
- 肃宗上元二年，皇后奏含晖院生金芝。（《同上》）
- 梁大同九年，金芝二十八茎生平水署，少府卿萧介以闻。（《大同起居注》）
- 王者德至于山，则芝实茂。（《白虎通》）
- 黄帝以五芝为房宫。（《张历》）
- 食芝延年不终，与真人同。（《瑞令记》）
- 汉建初三年，零陵泉陵县女子博宁宅内生芝草五本，茎叶紫色。（《论衡》）
- 芝草一年三华，食之令人眉寿度世，栽紫芝者其栽如豆。（《并同上》）

- 【集】武帝之世，芝房宝鼎之歌荐于郊庙。（班固《西都赋序》）
- 魏文帝青龙元年，神芝产于长平之习阳、许昌、弘农，中郎蒋充奉表以闻，其色丹紫，其质光耀，上别为三干，分为九枝，散为三十六茎，妥绥连属，有似珊瑚之状，考图按谱，盖美乎前代矣。（缪袭《神芝赞》）
- 濯灵芝于朱柯。（张平子赋）
- 玉芝含秀而晨敷。（孙绰赋）
- 煌煌灵芝，一年三秀。（嵇康诗）

#### 核心要点

1. **祥瑞象征**：芝草为王者慈仁之应，德至草木则生。
2. **五色分类**：紫芝、玄芝、金芝、玉芝，各随德感而生。
3. **仙药功效**：食芝延年度世，与真人同寿。
4. **应德而现**：宠老养贤、德政清明则芝草岁月不绝。

### 4.2 蓂荚

#### 原文（断句）

- 【史】蓂荚生于阶下，随月盈亏，依历开落。（《晋天文志》详见仪象类）

- 【传记】尧时瑞草号蓂荚，生于阶下，每朔日生一叶，月半至十五叶，自十五日已后日落一叶，月终而尽落，月小尽则馀一叶厌而不落。（《帝王世纪》）
- 蓂荚者，叶圆而五色，一名历荚，圣明之瑞也。（《孙氏瑞应图》）
- 周公辅幼主不矜，则蓂荚生。（《尚书大传》）
- 周公作乐而治，蓂荚生。（《尚书中候》）
- 老人星临国，则蓂荚生。（《春秋运斗枢》）
- 箕星得，则蓂荚生。（《同上》）
- 王者德至于地，则蓂荚生。（《孝经援神契》）
- 按孝经说，古太平蓂荚生阶，其味酸，王者取以调味，后以醯醢代之。（《风俗通》）
- 黄帝之时，有蓂荚者瑞草也，盖神灵之嘉应也。（徐整《正历》）
- 王者考历得其分度，则蓂生于阶。荚者树名也。（《白虎通》）

#### 核心要点

1. **历法象征**：蓂荚随月盈亏依历开落，朔生望落，为天然历法。
2. **圣明之瑞**：叶圆五色，王者德至于地、考历得度则生。
3. **辅政不矜**：周公辅幼主不矜、作乐而治则蓂荚生。
4. **调味之用**：古太平时味酸可调味，后以醯醢代之。

- **功能象义**：历法象征、圣明之瑞、辅政不矜、调味之用。
- **必要条件**：理解蓂荚的历法本质与圣德感应功能，在命理中识别蓂荚象征与王者考历辅政的关系。

### 4.3 连理木

#### 原文（断句）

- 【史】汉武帝幸雍，得奇木枝旁出复合，上异之。终军曰：「众枝内附，示无外也，殆将有削左衽而蒙化者。」（《汉书》）
- 明帝永平十七年，树枝内附，西南夷慕义贡献，西域诸国遣子入待。（《后汉书》）
- 宋文帝元嘉中，有谷栎二树连理。（《宋书》）
- 盩厔县献连理树，植之宫庭。（《隋书》）
- 贞观十八年，山南献木连理，交错玲珑，有同罗目，一丈之干并枝者二十馀所。（《唐书》）
- 又二十一年，玉华宫李树连理，隔阔合枝。（《同上》）

- 【传记】王者德化洽，八方合为一家，则木连理。（《孙氏瑞应图》）
- 王者不失民心，则木连理。（《同上》）
- 王者德至草木，则木连理。（《白虎通》）
- 连理者，或枝还合，或两树合拱。（《晋中兴证祥说》）

- 【集】皇树嘉德，风靡云披，有木连理，别干同枝，将承大同，应天之规。（曹植《魏德论》）
- 司空咸宁王尹蒲之七年，连理生于河之东邑。野夫来告，且曰：「吾不知古，殆气之交畅也。维吾王之德，交畅者有五。（云云）殊本连理之柯，同荣异垄之禾，吾睎之产兹土也久矣。愿记颂词，长言之于康衢。」颂曰：「木何为兮此祥？洵厥美兮在吾王。愿封植兮永固，俾斯人兮不忘。」（韩文，咸宁王浑瑊也）

#### 核心要点

1. **德化一统**：众枝内附，八方合为一家，王者德化洽则生。
2. **不失民心**：王者不失民心、德至草木则木连理。
3. **枝干合拱**：或枝还合、或两树合拱，象征同心同德。
4. **夷狄归化**：树枝内附应西南夷慕义、西域遣子入待。

#### 语义提取（L2）

- **自然属性**：连理木、枝旁出复合、众枝内附、交错玲珑、隔阔合枝、枝还合、两树合拱、殊本连理之柯。
- **功能象义**：德化一统、不失民心、枝干合拱、夷狄归化。
- **必要条件**：理解连理木的德化本质与一统象征，在命理中识别连理木象征与王者德政民心的关系。

### 4.4 朱草

#### 原文（断句）

- 【子】太清之世，四时不失其序，日月扬光，五星循轨，则朱草生。（《淮南子》）
- 朱草状似小桑，才长三四尺，枝叶皆赤，茎如珊瑚，生名山岭石之下。刻之汁流如血，以玉及金石银投其中，立可为丸泥；以金投其中，名曰金浆；以玉投之，名曰玉浆；服之皆长生也。（《抱朴子》）
- 圣人至德，上及太清，下及太宁，中及万灵，则朱草生。（《鹖冠子》）

- 【史】光武时，朱草生于水涯。（《后汉书》）
- 武帝时，朱草生。（《晋书》）

- 【传记】朱草，百草之精也，圣人之德无所不至则生。（《孙氏瑞应图》）
- 又曰：王者德无所不通，四方有歌咏之声。（《大戴礼》）
- 王者德下究地之原，则朱草生。（《尚书大传》）
- 帝命周公践祚，朱草畅生。（《同上》）
- 文命咸得俊乂在官，则朱草在郊。（《尚书中候》）
- 尧德清平，比隆伏羲，故朱草生郊。（《同上》）
- 君乘火而王，其政讼平，则地生朱草。（《礼斗威仪》）
- 王者德洽于地，则朱草生，食之令人不老。（《春秋感精符》）
- 枢星正，则朱草生。（《运斗枢》）
- 君劝农事，无夺民时，使之岁不过三日，行什一之税，进经术之士，开通障塞，恩及草木，则朱草生。（《繁露》）
- 王者德至草木，则朱草生。（《孝经援神契》）
- 朱草赤色也，可以染绛，别成黼黻之服，则为尊卑之差。（《白虎通》）
- 魏文帝欲受禅，朱草生于文昌殿侧。（《魏略》）

2. **百草之精**：朱草为百草之精，王者德无所不至则生。
3. **长生仙药**：服朱草金浆玉浆可长生不老。
4. **政平农安**：政讼平、劝农事、恩及草木则朱草生。
