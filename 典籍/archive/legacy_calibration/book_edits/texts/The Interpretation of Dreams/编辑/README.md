## Key Concepts to Extract

### Psychoanalytic Framework
- **Manifest/Latent**: Surface dream vs hidden meaning
- **Wish-Fulfillment**: Every dream fulfills an unconscious wish
- **Dream-Work**: Transformation processes from latent to manifest
- **Censorship**: Repression mechanism distorting dream content

### Core Mechanisms
- **Condensation** (凝缩): Multiple thoughts compressed into single image
- **Displacement** (移置): Emotional intensity shifted from important to trivial elements  
- **Symbolization** (象征化): Abstract thoughts represented through concrete symbols
- **Secondary Revision** (润饰): Logical narrative imposed post-dream
