## Key Ptolemaic Concepts

### Cosmological Framework
- **Æther** (以太): Fifth essence, encompassing sublunary realm
- **Ambient** (环境气层): Æthereal influence pervading atmosphere
- **Sublunary Elements**: Fire, Air, Water, Earth (四元素)
- **Configuration** (相位): Relative aspects between heavenly bodies

### Planetary Classifications
- **Benefics/Malefics** (吉星/凶星): Jupiter/Venus vs Saturn/Mars
- **Masculine/Feminine** (阳性/阴性): Active/passive qualities
- **Diurnal/Nocturnal** (日间/夜间): Solar/lunar affinity
- **Planetary Natures**: Hot/cold, moist/dry combinations

### Sign System
- **Tropical/Equinoctial** (回归/分点): Cardinal signs initiating seasons
- **Fixed**: Stable, sustaining signs
- **Bicorporeal/Mutable** (双体/变动): Transitional signs
- **Masculine/Feminine**: Positive/negative polarity

### Essential Dignities
- **Domicile** (庙旺/宫主): Planetary rulership of signs
- **Exaltation** (耀升): Maximum planetary strength
- **Triplicity** (三分主星): Elemental rulership
- **Terms/Bounds** (界限): Degree-based dignity
- **Face/Decan** (面/旬): 10-degree divisions
