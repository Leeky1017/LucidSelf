# 精校深度级别定义 V3.0

> **版本**: 1.0 | **创建日期**: 2026-01-27
>
> 本文件定义 LucidSelf 典籍精校的四个深度级别及其适用场景、模板要求、质量标准。

---

## 1. 深度级别概览

| 级别 | 名称 | 适用对象 | 工作量 | 质量目标 |
|:---|:---|:---|:---|:---|
| **Scholarly** | 学术级 | 核心概念（P0） | 高（4-8小时/卡） | 可被学术论文引用 |
| **Deep** | 深度级 | 主干概念（P1） | 中（1-2小时/卡） | 可被专业从业者参考 |
| **Standard** | 标准级 | 补充概念（P2） | 低（15-30分钟/卡） | 可被LS引擎消费 |
| **Index** | 索引级 | 索引条目（P3） | 极低（5分钟/卡） | 可定位和检索 |

---

## 2. Scholarly 级（学术级）

### 2.1 适用场景

- 各体系的**核心基础概念**（如八字的阴阳生克、占星的行星象征、荣格的原型理论）
- **跨体系桥接节点**（如官杀与土星相位的对应关系）
- 存在**学术争议**需要深入分析的概念
- 计划用于**对外发布、学术引用**的内容

### 2.2 模板要求

**必须完成所有 L1-L6 层次的所有字段**：

| 层次 | 必填内容 |
|:---|:---|
| S 综述 | 全部字段 + 核心论点一句话 |
| L1 深度理解 | L1.1 原文与上下文（完整）、L1.2 历史背景与学术脉络（详尽）、L1.3 作者意图与修辞分析 |
| L2 批判性分析 | L2.1 论证逻辑评估、L2.2 与至少2本其他典籍对比、L2.3 局限性与适用边界（详尽） |
| L3 语义本质 | L3.1 核心概念解构（每个概念都需四列完整）、L3.2 抽象规则提取（伪代码形式，包含UNLESS）、L3.3 因子提取（需论证） |
| L4 跨体系桥接 | L4.1 功能等价分析（至少2个目标体系）、L4.2 文化语境转译、L4.3 中立语义标签映射（需论证） |
| L5 验证 | L5.1 至少2个历史案例验证、L5.2 现代应用测试设想、L5.3 至少2条学术文献核验 |
| L6 元反思 | L6.1 自我审视（三维度）、L6.2 不确定性标注（五层次）、L6.3 至少3个待研究问题 |
| 附录 | A1 网络核对记录（完整）、A2 修订历史 |

### 2.3 质量验收标准

- [ ] 能清晰复述原文意图，无曲解
- [ ] 历史背景和学术脉络有据可查（有引用来源）
- [ ] 与至少2本其他典籍进行了实质性对比分析
- [ ] 明确了适用边界和已知反例
- [ ] 规则可操作（能写出可执行的伪代码）
- [ ] 跨体系映射有学术依据或严谨的逻辑论证
- [ ] 至少2个案例验证（历史或典籍中的案例）
- [ ] 引用了至少2条学术文献或经典著作
- [ ] 诚实标注了各层次的不确定性
- [ ] 可被学术论文作为参考资料引用

## 3. Deep 级（深度级）

### 3.1 适用场景

- 各体系的**主干概念**（如八字的用神、十神详解；占星的宫位详解；梦学的常见象征）
- 需要一定深度理解但**不涉及跨体系复杂映射**的概念
- 规则清晰、**争议较少**的内容

### 3.2 模板要求

**必须完成 L1 + L3 + L4 + L5（简化版）**：

| 层次 | 必填内容 | 可省略内容 |
|:---|:---|:---|
| S 综述 | 全部字段 + 核心论点一句话 | - |
| L1 深度理解 | L1.1 原文与上下文、L1.2 历史背景（简要） | L1.3 可省略或简化 |
| L2 批判性分析 | - | **可省略整层**，除非存在明显争议 |
| L3 语义本质 | L3.1 核心概念解构（可简化"现代重构"列）、L3.2 规则提取（可省略UNLESS）、L3.3 因子提取（简要论证） | - |
| L4 跨体系桥接 | L4.1 功能等价（1个目标体系即可）、L4.3 中立标签（简要论证） | L4.2 可省略 |
| L5 验证 | L5.1 至少1个案例验证 | L5.2、L5.3 可省略 |
| L6 元反思 | L6.2 不确定性标注（简化） | L6.1、L6.3 可省略 |
| 附录 | A2 修订历史 | A1 可省略 |

### 3.3 质量验收标准

- [ ] 原文理解准确，上下文清晰
- [ ] 规则可操作（能指导实际应用）
- [ ] 至少1个跨体系映射
- [ ] 至少1个案例验证
- [ ] 可被专业命理/占星/心理咨询从业者作为参考

### 3.4 适用典籍章节示例

| 典籍 | 适用章节 |
|:---|:---|
| 子平真诠 | 论十神各章（正官论、七杀论等） |
| 滴天髓 | 天干论各干详解 |
| The Inner Sky | Signs 各星座详解 |
| Archetypes | 各原型详解章节 |

## 4. Standard 级（标准级）

### 4.1 适用场景

- **补充性内容**（如神煞、次要格局、低频梦象）
- 内容**相对简单明确**，无需深度分析
- 主要用于**扩大覆盖面**而非深化理解

### 4.2 模板要求

**必须完成 L1（简化）+ L3（核心）**：

| 层次 | 必填内容 | 可省略内容 |
|:---|:---|:---|
| S 综述 | card_id、semantic_unit、source_range、priority | depth_level 默认为 Standard |
| L1 深度理解 | L1.1 原文（必须）、上下文（一句话即可） | L1.2、L1.3 省略 |
| L2 批判性分析 | - | **省略整层** |
| L3 语义本质 | L3.2 规则提取（IF-THEN 形式即可）、L3.3 因子提取（无需详细论证） | L3.1 可省略 |
| L4 跨体系桥接 | L4.3 中立标签（无需论证） | L4.1、L4.2 省略 |
| L5 验证 | - | **省略整层** |
| L6 元反思 | - | **省略整层** |

### 4.3 质量验收标准

- [ ] 原文准确无误
- [ ] 因子提取合理
- [ ] 规则可被引擎解析
- [ ] 中立标签映射基本合理

### 4.4 适用典籍章节示例

| 典籍 | 适用章节 |
|:---|:---|
| 三命通会 | 神煞详解、次要格局 |
| 周公解梦 | 低频梦象条目 |
| 梦林玄解 | 冷门门类 |
| 塔罗典籍 | 小阿卡那详解 |

## 5. Index 级（索引级）

### 5.1 适用场景

- **仅需保留索引**的内容（如序言、版本说明、重复性条目）
- 内容**无法提取有意义的规则或因子**
- 但需要**保留定位能力**以便后续查阅

### 5.2 模板要求

**仅需 S 综述**：

| 字段 | 必填 | 说明 |
|:---|:---|:---|
| card_id | 是 | 格式：`<book_abbr>_<chapter>_idx_<seq>` |
| semantic_unit | 是 | 简述内容（如"序言""版本说明"） |
| source_range | 是 | 精确定位 |
| depth_level | 是 | 固定为 `Index` |
| priority | 是 | 固定为 `P3` |

### 5.3 质量验收标准

- [ ] 定位准确，可通过 source_range 找到原文
- [ ] semantic_unit 简明描述了内容性质

## 6. 深度级别选择决策树

```
开始
  │
  ▼
是否为各体系核心基础概念？───是──▶ Scholarly
  │
  否
  ▼
是否涉及跨体系复杂映射或学术争议？───是──▶ Scholarly
  │
  否
  ▼
是否为各体系主干概念？───是──▶ Deep
  │
  否
  ▼
是否能提取有意义的规则/因子？───是──▶ Standard
  │
  否
  ▼
是否需要保留索引以便查阅？───是──▶ Index
  │
  否
  ▼
不纳入精校范围
```

## 7. 各典籍章节的深度级别建议

| 典籍 | Scholarly | Deep | Standard | Index |
|:---|:---|:---|:---|:---|
