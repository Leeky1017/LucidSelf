> Each zodiac sign is not merely a character type but an evolutionary method—a pattern of growth involving specific strategies for development, inherent resources to draw upon, and shadow expressions that emerge when growth is resisted.

### L3.2 Abstract Rule Extraction

**Rule ID**: `rule_inner_sky_sign_interpretation`

**Rule Structure** (pseudo-code form):

```
FOR each sign S:
  S.developmental_task = primary_growth_challenge(S)
  S.strategy = experiential_path_that_evokes(S.developmental_task)
  S.resources = native_capacities_supporting(S.strategy)
  S.shadow = dysfunction_when_resisting(S.developmental_task)
  
WHEN interpreting sign_placement(planet P in sign S):
  IF individual is growing:
    RETURN P.function expressed through S.resources pursuing S.strategy
  IF individual is resisting:
    RETURN P.function expressed through S.shadow
WITH confidence = High
BECAUSE "Read them as methods of growth and everything is transformed"
```
