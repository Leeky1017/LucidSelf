# geocoding-first-class — Intent Alignment

## 目的（可验收语言复述）

`geocoding-first-class` 规格用于把 LS 的「地理编码」能力固化为可商用、可审计契约：定义标准化地理实体、精度/置信度元数据、回退记录与指标报表的最小产物与标识符要求，并确保每次地理编码结果可追溯到 `version_id/user_id`（以及 `engine_id/corpus_release_id` 如适用），从而为 Gate-3 的“可控精度 + 可解释回退 + 成本/质量可观测”提供可执行、可阻断、可复现的验收口径。

## 边界（不做什么）

- 本 spec 仅定义 MUST/SHALL 的契约、字段语义、门禁与验收口径。
- 本 spec 不规定具体实现细节、代码路径、技术选型或伪代码。

## 成功条件（可判定）

- 关键产物（标准化地理实体、精度/置信度元数据、回退记录、指标报表）具备字段级约束与完整性语义。
- 对外契约字段与拒绝原因以机器可读形式固定，并可通过命令验证。
- Gate-3 的验收口径可执行；失败具备明确阻断策略并可追溯到 `version_id/user_id`。
