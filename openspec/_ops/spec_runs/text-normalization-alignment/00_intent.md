# text-normalization-alignment — Intent Alignment

## 目的（可验收语言复述）

`text-normalization-alignment` 规格用于把 LS 的「文本归一化与对齐」资产能力固化为可审计契约：定义归一化规则集、对齐映射、变换审计记录、覆盖率与例外清单的最小产物与标识符要求，并确保所有资产决策可追溯到 `corpus_release_id` 与 `version_id`，满足资产门禁的“许可/血缘/QA/冲突处置可追溯”与可执行验收口径。

## 边界（不做什么）

- 本 spec 仅定义 MUST/SHALL 的契约、字段语义、门禁与验收口径。
- 本 spec 不规定具体实现细节、代码路径、技术选型或伪代码。

## 成功条件（可判定）

- 关键产物（规则集、对齐映射、变换审计记录、覆盖率与例外清单）具备字段级约束与完整性语义。
- 对外契约字段与拒绝原因以机器可读形式固定，并可通过命令验证。
- 资产门禁验收口径可执行；失败具备明确阻断策略并可追溯到 `corpus_release_id`/`version_id`。
